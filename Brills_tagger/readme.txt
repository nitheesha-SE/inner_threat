This file includes the brillspos tagger implementaion with the file name brillscomplete.py.
The input data for the file is present in the folder data with train data name hw3_train and the test data with the name hw3_heldout.


This code can be executed with the command:
   Python brillscomplete.py data/hw3_train data/hw3_heldout

Make sure that the train and the test data is present in the folder data.
